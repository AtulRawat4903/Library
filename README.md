# 📚 Library

A simple library built with **HTML, CSS, and JavaScript** as part of **The Odin Project**. The application allows users to add books, remove books, and update their reading status through a clean, responsive interface.

---

## 🚀 Live Demo

https://atulrawat4903.github.io/Library

---

## ✨ Features

- 📖 Add new books using a modal dialog
- 🗑️ Remove books from the library
- ✅ Toggle a book's read/unread status
- 🆔 Unique ID assigned to every book using `crypto.randomUUID()`
- 🎴 Responsive card-based layout
- 🌙 Modern dark-themed UI
- 📱 Responsive design for different screen sizes
- 📂 Empty library state for better user experience

## ⚙️ How It Works

### Add a Book

- Click the **New Book** button.
- Fill in the book details.
- Submit the form to add the book to the library.

### Toggle Read Status

- Click **Mark as Read** or **Mark as Unread** to update the book's reading status.

### Remove a Book

- Click the **Remove** button to delete the selected book from the library.
